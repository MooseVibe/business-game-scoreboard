# Бизнес игра

HTML/CSS/JS табло для бизнес-игры на четыре команды.

Можно открыть локально через `index.html` или использовать опубликованную версию на GitHub Pages.

## Дополнительные инструменты

- `interview-assistant/` — статический прототип ассистента интервьюера для IQ-Provision. Лежит отдельно от бизнес-игры и не влияет на её код.
- `interview-assistant/supabase-interview-assistant.sql` — схема для синхронизации ассистента через Supabase, если понадобится работа с разных устройств.
